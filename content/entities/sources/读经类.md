# 读经类

**读经类**是主恢復中關於讀經方法、聖經要道、結晶讀經和聖經各卷概論的信息合集，共計**43篇信息**，涵蓋多本重要著作。

## 定義

读经类是主恢復中關於讀經方法、聖經要道、結晶讀經和聖經各卷概論的信息合集，幫助信徒建立穩固的聖經知識基礎。

## 概述

读经类信息的核心目標是：
- **掌握讀經方法**——學習正確有效地讀聖經
- **認識聖經全貌**——了解聖經的整體結構和主要路線
- **深入經卷要義**——掌握各卷書的中心思想和要道
- **結晶讀經**——從經卷中萃取出最精華的真理結晶

這些信息幫助信徒建立穩固的聖經知識基礎，是深入研讀聖經的重要入門和指引。

## 讀經的層次

读经类信息涵蓋多個層次：
1. **方法層**——讀經之路、一年一遍等實用方法
2. **概覽層**——聖經提要、十條路線等整體認識
3. **要道層**——聖經要道、各卷概論等真理認識
4. **結晶層**——結晶讀經系列，萃取經卷精華
5. **應用層**——從基督與召會觀點看聖經
