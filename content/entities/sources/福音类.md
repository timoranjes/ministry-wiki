# 福音类

**福音类**是主恢復中關於傳揚福音和救恩真理的信息合集，共計**70篇信息**，涵蓋多本福音單張、小冊和信息集。

## 定義

福音类是主恢復中關於傳揚福音和救恩真理的信息合集，涵蓋從基本的福音真理到深入的救恩啟示，是傳福音和幫助初信者的重要資源。

## 概述

福音类信息的核心目標是：
- **認識神**——幫助未信者認識神的存在和屬性
- **認識救恩**——明白基督的救贖和重生之道
- **接受救恩**——引導人接受基督作救主
- **得救的確據**——幫助新得救者確知自己已得救

這些信息涵蓋從基本的福音真理到深入的救恩啟示，是傳福音和幫助初信者的重要資源。

## 福音的層次

福音类信息涵蓋多個層次：
1. **入門層**——回答「有神嗎？」「耶穌是誰？」等基本問題
2. **救恩層**——闡明人的罪、基督的救贖、重生與得救
3. **確據層**——幫助信徒確知得救、明白救恩的保障
4. **深化層**——神經綸的福音、變死亡為生命等更深啟示
